# ViaCEP MCP Server

Servidor MCP para consulta de CEPs via API ViaCEP.

## Instalação

```bash
poetry install
```

## Execução

```bash
poetry run mcp-viacep
```

## Ferramenta Disponível

- `buscar_cep(cep)`: Busca informações de endereço pelo CEP

## Configuração no Amazon Q

**Nome:** mcp-viacep  
**Comando:** poetry  
**Args:** run mcp-viacep  
**Diretório:** c:\devtools\mcp-viacep
